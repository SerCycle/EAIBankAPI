# EAIBankAPI
## Installation
- Clone first this repository
- and run `$ npm install`

## Endpoint
### User
> User Insert (POST)
```
https://eai-bank-api.glitch.me/users
```
> User List (GET)
```
https://eai-bank-api.glitch.me/users
```
> User Single Selected (GET)
```
https://eai-bank-api.glitch.me/users/:id
```
